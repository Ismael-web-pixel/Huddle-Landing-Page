##     Huddle Landing Page
##     Description

L’objectif est de reproduire fidèlement la mise en page et le design proposés, en utilisant uniquement HTML et CSS, tout en assurant une compatibilité responsive sur différents écrans (desktop, tablette, mobile).



##   Fonctionnalités

Page web entièrement responsive

Structure sémantique claire et organisée


Utilisation des polices Poppins et Open Sans

Icônes intégrées via Font Awesome

Boutons interactifs avec effets de survol (hover)



##  Technologies utilisées

HTML5

CSS3 (Flexbox, Media Queries)

Font Awesome (pour les icônes)
